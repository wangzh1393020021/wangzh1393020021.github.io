第二章：类和对象
1.面向过程
面向过程就是会把事情拆分成一个个函数和数据，然后按照一定的顺序，执行这些方法，等方法执行完，事情就搞定啦。
2.面向对象
是符合人类思考习惯的一种编程思想，在编程的时候，是由编程者变成指挥者，简单来说就是给对象赋一些属性和方法，然后让每个对象执行自己的方法，解决问题。
3. 程序的执行过程
①将字节码加入内存；
②对字节码进行合法性检查；
③jvm会为每个字节码文件都生成一个对象(class)；
④执行静态代码块，初始化静态成员等；
⑤执行入口函数main方法；

5.什么是类
类：是具有相同数据成员和函数成员的一组对象的集合，它为属于该类的全部对象提供了抽象的描述。
这个世界上，所有东西都可以理解为对象，万物皆对象。而类是指对一些对象的总称，对象构成了类，打个比方，人类是类，你是对象，也就是说，对象是独一无二的。

第三单元：FeiGe快递系统-封装
封装
什么是封装：封装是指讲 对象的状态信息隐藏在对象内部。
封装的概述：被private修饰的成员，就被 类 私有化。被私有化的成员，不能 再 类的外面访问 ，只能在类的内部访问。
封装的好处：1.提高安全性2.提高重复性3.隐藏实现细节
封装的使用步骤
1.用private修饰属性(成员变量)。
2.提供set，get方法，set方法完成对属性的赋值，get方法，获取属性的值。
3.加入控制语句。
Integer:封装类
int Integer
char Char

double Double
float Float
这些都是基本数据类型，对应的封装类，在存储的时候，先把基本数据类型的值转化为对象，真正使用的时候又转换为基本数据类型，这个过程称之为自动装箱自动拆箱。

第5单元FeiGe快递系统-多态
1.1什么是多态：
1.可以理解为，一个对象可以有多种形态。
2.多态的前提:继承、重写、父类引用指向子类的对象。
具体方法就是：向上转型、向下转型。
向下转型：
语法
<父类型><引用变量名>=new<子类型>();
向上转型
语法：
<子类型><引用类型>=（<子类型>）<父类型引用变量>

有什么用途：

1.父类作为参数。
如果方法的参数是类类型，那么调用的方法传参时传的参数就是对应的对象，或者他的子对象，如果传子类对象，就发生啦向上转型。
一旦向上转型那么只能调用得到父类声明过方法，不能调用得到自己独有的方法。
2. 父类作为方法的返会值类型

第6单元FeiGe快递系统-抽象
被abstract修饰的类称为抽象类
public abstract class 类名{}
抽象类与普通类的区别：
1.抽象类没有构造方法，导致抽象类不能创建对象
2.抽象类中可以写抽象方法
抽象方法：被abstract修饰的方法称为抽象方法
public abstract 返回值类型 方法名（参数列表）；
抽象方法没有方法体，只有方法的声明。
没有子类的抽象方法是没有意义的。
继承抽象类必须重写抽象类中所有的抽象方法。
2.2final修饰符
1.被final修饰的类不能被继承
2.被final修饰的方法不能被重写
3.被final修饰的变量是常量

语法：

接口的作用，仅仅是，把项目需要完成的方法，先统一声明下。

IO流
流是一个形象的说法，文件中的数据像水一样流向java代码，或者和Java代码中的数据，像水一样流向文件，操纵这种流动的类，统称为IO流。
既然是流动，就有流入，流出
流的分类：
按方向分：输入流，输出流
按一次操作的字节个数分：字节流，字符流
要讲的流如下：
输入字节流
InputStream
