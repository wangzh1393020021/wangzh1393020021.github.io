#mysql第二次课
表的设计三大泛型
第一范式
表中的字段都是不可分割的最小数据单元。
第二范式
每张表只能描述一件事情，并且每张表必须有主键。
第三泛型
非主键字段必须依赖于主键字段，换句话说，非主键要与主键字段有直接关系。
注意：设计表时，尽可能的符合规范，但是有时为了性能，可以牺牲一点规范。
二、mysql中使用运算符
算术运算符
+ - * /
逻辑运算符
and or
比较运算符
< > >= <= != <>

三、聚合函数
count() 求行数
sum() 求和

    #算出goods表中商品总价
    SELECT SUM(gprice) '商品总价格' FROM goods;

avg()  求平均值

    #算出goods表中商品的平均价格
    SELECT AVG(gprice) '商品平均价格' FROM goods;

max()  求最大值

    #查询出价格最高的商品信息
    SELECT * FROM goods WHERE gprice =(SELECT MAX(gprice)'最高价格' FROM goods);

min()  求最小值

四、几个关键字
in 在什么什么范围内 例如 name in（‘张三’，‘李四’）

    #查询出商品名称是小熊雨伞，外套的都查询出来
    SELECT * FROM goods WHERE gname IN('小熊雨伞','外套');

not in 
 name not in （‘张三’，‘李四’）姓名字要不是张三李四即可以查询出来

is null   判断数据是否为空

is not nul     是否不为空

like  模糊查询
limit 查询表中的前几行
    #查询goods表中前两行
    #limit 0,2; 从下标是0的行开始取两行。
    SELECT * FROM goods LIMIT 0,2;

between…and…   在什么什么范围内
    #查询出价格处于10到50的商品价格的商品信息
    SELECT * FROM goods WHERE gprice BETWEEN 10 AND 50;
distinct 去重
    #把商品名字去重
    SELECT DISTINCT gname FROM goods;
